About

This website uses the PG County Library API. Below are instructions on how to utilitze the website's features.

Load County Button

  • Before generating a list of random libraries, you need to load the data from the API. Click this button first before proceeding.

    • Generate List Button

  • After hitting the, "Load County Button", click this button to generate a list of 15 libraries in PG County. As you generate the list, you will see the list below and markers displayed on the map. The map panes to the marker corresponding to the first library in the list. The API contains a large list of libraries, but the button will only generate 15 libraries.

    • Clear County Data

  • If you want to reset and refresh your data. This button will clear the list of libraries you have. You can then click on generate button again to make another list of libraries.

    • Filter by Library Name

  • Type the name of a library you're looking for and the list will filter down to results based on what you have typed.

    • Filter by Zip Code

  • Similar to the name filter, this will filter down the list to libraries that are in the same zip code you input.

    • Display Chart

  • This website contains a chart of libraries per zip code. Click on it and the chart will appear.

    • Display Map

  • This will go back to the map display.
